Problem: Bike Class with State Management

Description:
This program implements a Bike class that manages speed and fuel level
using accelerate and decelerate methods.

Files:
- Bike.h    : Class declaration
- Bike.cpp  : Class implementation
- main.cpp  : Driver program
- Makefile  : Build instructions

Compilation:
make

Execution:
./bike

Clean:
make clean

Platform:
Linux (Ubuntu)

Compiler:
g++
