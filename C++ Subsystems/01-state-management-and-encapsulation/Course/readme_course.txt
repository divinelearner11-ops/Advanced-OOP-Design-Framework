Problem: Course Class with Constructor Chaining

Description:
This program implements a Course class using constructor chaining.
Default values for credits and instructor are assigned using chained constructors.

Files:
- Course.h    : Class declaration
- Course.cpp  : Class implementation
- main.cpp    : Driver program
- Makefile    : Build instructions

Compilation:
make

Execution:
./course

Clean:
make clean

Platform:
Linux (Ubuntu)

Compiler:
g++
